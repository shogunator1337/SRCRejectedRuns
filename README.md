# Speedrun.com Rejected Runs Extension

Расширение для браузера, которое добавляет возможность просматривать отклоненные (Rejected) и ожидающие проверки (Pending) спидраны прямо на странице лидерборда игры на сайте Speedrun.com.

## Функционал

- Прямо на странице лидерборда (если раны не найдены, например из-за фильтров или отсутствия проверенных ранов) появляется таблица с ожидающими и отклоненными ранами.
- При наличии ранов на лидерборде, таблица с отклонёнными/ожидающими добавляется ниже или интегрируется в список.
- Добавляются удобные фильтры "Pending" и "Rejected" рядом с остальными фильтрами.
- Позволяет модераторам и обычным пользователям удобно следить за статусом проверки спидранов, не переходя в раздел модерации.

## Готовые сборки (GitHub Releases)

Вы можете скачать уже готовые к установке файлы в разделе [Releases](../../releases) репозитория на GitHub:
- Для **Firefox** предоставлен готовый и подписанный файл `.xpi`, который устанавливается двойным кликом или перетягиванием файла на вкладку `about:addons`.
- Для **Chrome / Opera GX** можно скачать архив с расширением, распаковать его и установить как описано ниже.

> ⚠️ **Важное предупреждение для пользователей Chrome:** Так как расширение устанавливается из распакованной папки (не из Chrome Web Store), браузер Chrome время от времени может автоматически отключать его из соображений безопасности. В этом случае вам может потребоваться заново включить тумблер расширения на странице `chrome://extensions/` или переустановить его.

## Установка вручную (из исходного кода)

Вы можете установить это расширение локально (без публикации в магазин расширений).

### Google Chrome

1. Скачайте папку `extension` (исходный код расширения) на ваш компьютер.
2. Откройте Chrome и перейдите по адресу `chrome://extensions/`.
3. В правом верхнем углу включите **"Режим разработчика"** (Developer mode).
4. Нажмите кнопку **"Загрузить распакованное расширение"** (Load unpacked) в левом верхнем углу.
5. Выберите папку `extension`. Расширение будет установлено и готово к работе!

### Opera GX

1. Скачайте папку `extension` на ваш компьютер.
2. Откройте Opera GX и перейдите по адресу `opera://extensions/`.
3. В правом верхнем углу включите **"Режим разработчика"** (Developer mode).
4. Нажмите кнопку **"Загрузить распакованное расширение"** (Load unpacked).
5. Выберите папку `extension`. Расширение загрузится и активируется.

### Mozilla Firefox

1. Скачайте папку `extension` на ваш компьютер.
2. Откройте Firefox и перейдите по адресу `about:debugging#/runtime/this-firefox`.
3. Нажмите кнопку **"Загрузить временное дополнение..."** (Load Temporary Add-on...).
4. В открывшемся окне выберите файл `manifest.json` внутри папки `extension`.
5. Расширение будет установлено на время текущей сессии.

**Как упаковать и установить расширение в Firefox навсегда:**

Поскольку Firefox требует обязательной подписи расширений (кроме Developer/Nightly версий), для постоянного использования:
1. Зайдите в папку `extension`, выделите все файлы (включая `manifest.json`) и запакуйте их в обычный ZIP-архив (важно запаковать сами файлы, а не саму папку `extension`).
2. Перейдите на сайт [Firefox Add-on Developer Hub](https://addons.mozilla.org/ru/developers/) и авторизуйтесь (потребуется аккаунт Firefox).
3. Нажмите **"Submit a New Add-on"** (Отправить новое дополнение).
4. На вопрос о том, как вы хотите распространять дополнение, выберите **"On your own"** (Самостоятельно/Не публиковать в магазине).
5. Загрузите ваш ZIP-архив. Система автоматически проверит его и выдаст подписанный файл формата `.xpi`.
6. Скачайте этот `.xpi` файл, откройте Firefox и перейдите в `about:addons`. Перетащите скачанный файл в окно браузера (или выберите "Установить дополнение из файла..." в меню настроек дополнений) для постоянной установки.
